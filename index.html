Executive Summary

Public Landing Page

A welcoming entry point for men curious about The Men's Circle, with clear pathways to learn more and connect.

Member Resource Hub

A simple authenticated area where active men can access directories, documentation, and announcements.

Google Sheets Backend

Administrators continue working exactly as they do now. The website automatically reflects current information.

The Vision

For the Public

  • Clean, inviting landing page explaining who we are and what we offer
  • Clear contact pathways to reach the right men
  • Information about how to get involved or attend events

For Active Men

  • Member Directory — Names, contact info, circle affiliation, status
  • Documentation Library — Agendas, guidelines, resources
  • Announcements — Current information for the community

Simple Login: Magic Links

A man enters his email, receives a link, clicks it, and he's logged in. No passwords to remember or reset. Simple, secure, minimal friction.

If your email is in the roster sheet, you have access. Remove the email, access is revoked. The sheet IS the access control.

Why Google Sheets as the Backend?

We already maintain extensive information in Google Sheets — rosters, documentation, agendas, contact lists. This proposal builds on that investment rather than replacing it.

How It Works

  • Designated Google Sheets serve as the "database"
  • The website reads from these sheets automatically
  • When someone updates a sheet, the website reflects the change within minutes
  • No duplicate data entry, no sync issues, no new systems to learn

What This Means for Administrators

  • Continue using Google Sheets exactly as you do now
  • No new software to learn
  • No manual website updates needed for routine information changes
  • No security patches or plugin updates to manage

This Approach vs. WordPress

Consideration Google Sheets + Modern Web WordPress
Content Updates Edit sheets → automatic Login → edit → publish
Security Maintenance Automatic (Vercel handles) Manual updates required
Plugin/Theme Issues None Common problems
Member Directory Live from roster sheet Manual entry or plugin
Learning Curve None for content Moderate
Hosting Cost ~$0/month ~$10-30/month
Long-term Reliability Very stable Requires attention
Custom Code Changes Requires developer Requires developer

Advantages of This Approach

  • Minimal ongoing maintenance
  • Single source of truth in Google Sheets
  • Administrators keep their workflow
  • Modern, fast, secure
  • Cost-effective

Honest Considerations

  • Initial build requires development work
  • Code changes require technical skill
  • Less "off-the-shelf" than WordPress
  • Depends on sheet structure staying consistent
